This is a simple Contact Management Web Application built using the MERN stack.
The project was created as part of a web developer task to demonstrate full-stack fundamentals including frontend, backend, database integration, and deployment.

***Features***
Add a new contact with Name, Email, Phone, and Message

Client-side form validation

View all submitted contacts instantly (no page reload)

Delete contacts

Clean and responsive UI

Fully deployed on cloud

***Tech Stack***

**Frontend**

React.js

CSS

React Hooks (useState, useEffect)

**Backend**

Node.js

Express.js

Database

MongoDB Atlas (cloud database)

**Deployment**

Render (Backend + Frontend)

GitHub for version control
